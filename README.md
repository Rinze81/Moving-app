# Moving Mate

引っ越しの比較・タスク・総額管理を 1 つにまとめる、スマホ向けの静的 Web アプリです。

## 機能

- 物件ごとの初期費用・月額費用比較
- 不動産会社ごとの管理
- タスクのカテゴリ分け、期限管理、完了チェック
- 家具・家電・引っ越し業者などの追加費用登録
- 想定総額の自動計算
- 物件メモと画像保存
- CSV 出力
- PWA 対応

## ファイル構成

- `index.html`: 画面本体
- `styles.css`: モバイル優先のスタイル
- `app.js`: データ保存、UI 描画、CSV 出力、PWA 登録
- `manifest.webmanifest`: PWA マニフェスト
- `sw.js`: 最小のオフラインキャッシュ

## 起動方法

ブラウザで `index.html` を開くだけでも使えます。PWA と Service Worker を安定して使う場合はローカルサーバー経由を推奨します。

Windows 例:

1. このフォルダでシンプルな HTTP サーバーを起動する
2. `http://localhost:8000` のような URL を iPhone Safari で開く
3. Safari の共有メニューから「ホーム画面に追加」を選ぶ

ローカルサーバーの方法は利用環境に合わせて選んでください。
